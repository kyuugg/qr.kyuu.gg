A lightweight QR code generator with a clean, minimal interface. Enter any text or URL, generate a QR code instantly, and use it anywhere you need—without accounts, ads, or unnecessary distractions.

> 🌐 **[Generate QR codes → qr.kyuu.gg](https://qr.kyuu.gg)**

## ⚡ Highlights

- Generate QR codes instantly
- Supports text, URLs and any other input
- High-quality PNG output
- Minimal, distraction-free interface
- Fast client-side generation
- No backend or user data collection

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)
- QRCode.js

## 🚀 Getting Started

Clone the repository and open the project in your browser.

```bash
git clone https://github.com/kyuugg/qr.kyuu.gg.git
cd qr.kyuu.gg
open index.html
```

## 💡 About

This project is part of the **KyuuGG** ecosystem—a collection of lightweight web tools built while learning modern frontend development. The goal was to create a simple utility with a polished interface while keeping the codebase easy to understand and maintain.

As of **July 2026**, QR generation is handled entirely on the client using **QRCode.js**, meaning no data ever leaves the user's browser.
